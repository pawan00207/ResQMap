# ResQMap

ResQMap ek open-source web platform hai jo OpenStreetMap data ka use karke emergency services, accessibility aur disaster related information ek hi map par dikhata hai.

Is project ka goal ye hai ki users ko nearby important services easily mil sakein aur crisis situations me quick decision lene me help mile.

---

## Project Purpose

Urban areas me aksar emergency services, accessible routes aur local resources ki information alag-alag jagah milti hai.
ResQMap in sabko ek single interface me combine karta hai.

---

## Main Features

* Nearby hospitals, police stations aur fire stations locate karna
* Map par route dekhna
* Wheelchair friendly routes filter karna
* Local resources (pharmacy, shelters, vendors) dikhana
* Disaster related safe zones aur incident reporting

---

## Tech Stack

Frontend: React.js + Leaflet
Backend: Node.js + Express
Database: PostgreSQL (PostGIS)
Map Data: OpenStreetMap

---

## Project Structure

ResQMap/

* client → frontend code
* server → backend APIs
* database → schema/config

---

## How to Run

1. Repository clone karo
2. Dependencies install karo
3. `.env` file configure karo
4. Docker ya npm se project run karo
